# 📊 Apex Retail – Store Performance Analysis | October 2024

## 📌 Project Overview

This project showcases an interactive **Power BI dashboard** developed to analyze retail store performance during **October 2024**. The dashboard transforms raw sales data into meaningful business insights through KPIs, interactive slicers, and visualizations that help monitor sales performance, profitability, and regional trends.

The dashboard enables business users to quickly evaluate operational performance and identify opportunities for improvement through data-driven decision-making.

---

## 🎯 Business Objectives

- Monitor overall sales performance.
- Compare sales across different regions.
- Evaluate store profitability.
- Track operational efficiency using KPIs.
- Enable interactive filtering by region.
- Support business decision-making with visual analytics.

# 📊 Dashboard Visualizations

## 1. Sales Revenue by Region

**Chart Type:** Horizontal Bar Chart

Displays total sales revenue generated across different regions.

### Regions Included

- West
- Northeast
- Midwest
- Southeast

### Key Insight

- **West** achieved the highest sales revenue.
- **Southeast** recorded the lowest revenue during the reporting period.

---

## 2. Average Daily Sales by Region

**Chart Type:** Horizontal Bar Chart

Compares the average daily sales across all regions.

### Key Insight

- West maintained the strongest daily sales performance.
- Midwest and Northeast delivered consistent daily sales.
- Southeast showed opportunities for sales improvement.

# 🎛 Interactive Slicers

The dashboard includes dynamic slicers that allow users to filter data instantly.

### Region Slicer

Users can filter dashboard metrics by:

- All Regions
- Midwest
- Northeast
- Southeast
- West

Selecting a region automatically updates:

- KPI Cards
- Revenue Charts
- Average Daily Sales
